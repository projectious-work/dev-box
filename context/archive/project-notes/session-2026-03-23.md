## Key technical findings

- **Zellij must be foreground PTY process** to render. Can't background, can't parallel.
- **Tab cycling via embedded pane:** a 1-row hidden pane inside the layout runs
  `zellij action go-to-tab N` — works because the pane is inside the Zellij session.
- **Theme colors in player:** recordings contain ANSI color indices (0-7), but the player
  maps them via CSS `--term-color-N` variables. Overriding per theme makes them visually distinct.
- **OSC 10/11 doesn't help:** Zellij fills every cell, so terminal bg color is invisible.
- **`pkill -x zellij`** (exact name match) avoids killing asciinema/bash parents.
  `pkill -f` pattern matching was too broad and killed the recording process.
