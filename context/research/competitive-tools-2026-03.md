# Competitive Tools & Ecosystem — March 2026

## Vibe Coding Platform Convergence

### Replit Agent v4
Full AI-agent app builder. Agent writes code, sets up infra, tests, deploys from natural language. Up to 200-minute autonomous runs, parallel agents, built-in auth/database/hosting/monitoring. Mobile app (iOS/Android). Free tier; Core $20/month.

### Google AI Studio + Antigravity
"Full-stack vibe coding" (announced 2026-03-18). The Antigravity agent takes over from a text prompt — plans, writes multi-file code, runs tests, self-corrects. Firebase integration: auto-detects data persistence/auth needs. Supports React, Angular, Next.js. Powered by Gemini 3.1 Pro. Free for prototyping.

### Lovable — Business Operations Pivot
Pivoting from vibe-coding to business operations platform (2026-03-19):
- Data analysis: CSV/PDF processing with Python sandbox
- Document generation: reports, invoices, pitch decks
- File-to-app: spreadsheets → functional apps with databases + RBAC
- Pricing: Free (5 credits/day), Pro ($25/mo), Business ($50/mo + SSO), Enterprise

### OpenAI Super App
Planned desktop app merging ChatGPT, Codex, and Atlas browser. Led by Fidji Simo. Explicitly cites Anthropic/Claude enterprise success as "wake-up call."

## Pricing Convergence

All major platforms converge on consumption-based pricing: Lovable ($25/mo ~150 credits), Replit (compute-minute), Google (token-based via Gemini API). Credit/token model is the market expectation.
